Introduction

JavaScript est un langage de programmation populaire utilisé pour le développement web.

Il permet de rendre les pages web interactives et dynamiques.

Voici un exemple de code JavaScript simple :

console.log("Hello, world!");

Ce code affiche "Hello, world!" dans la console du navigateur.

Variables et Types

JavaScript utilise var, let et const pour déclarer des variables.

Chaque type de déclaration a ses propres caractéristiques :

  • var : portée globale ou fonction.
  • let : portée de bloc.
  • const : constante, ne peut pas être réassignée.

Exemple de déclaration de variables :

var x = 5;
let y = 10;
const PI = 3.14;
Fonctions

Les fonctions permettent de réutiliser du code et d'organiser la logique du programme.

Voici un exemple de déclaration de fonction :

function greet(name) { return "Hello " + name; }

On peut appeler cette fonction ainsi :

console.log(greet("Alice")); // Affiche "Hello Alice"
Boucles

JavaScript propose plusieurs types de boucles pour répéter des actions.

Les deux types de boucles les plus courants sont :

  • for : boucle avec compteur.
  • while : boucle conditionnelle.

Exemple de boucle for :

for (let i = 0; i < 5; i++) {
  console.log(i);
}
Objets et Tableaux

Les objets et tableaux sont des structures de données essentielles en JavaScript.

Exemple d'objet :

let person = { name: "Alice", age: 25 };

Exemple de tableau :

let fruits = ["pomme", "banane", "orange"];

On peut accéder aux éléments d'un objet ou d'un tableau ainsi :

console.log(person.name); // Affiche "Alice"
console.log(fruits[0]); // Affiche "pomme"
